# 登录接口 /login

- 登录网站需要在这个接口进行登录认证
- 需要用户提供一下几个参数
  - username: 用户名
  - userpasswrod: 用户密码
  - cookie:保存认证数据
 
 
 # 注册接口 /registe
 - 用户注册需要在这里进行，因为自己玩的小网站所以就没加短信认证之类的东西
 - 请求方式：POST
 - 参数
   - username: 用户的帐号名，仅支持数字字母下划线
   - userpassword：用户密码,仅支持数字字母下划线
   - email： 电子邮箱目前仅支持163和qq邮箱
   - phone: 手机号码
   - name: 用户名字，没有输入的话默认为用户帐号
